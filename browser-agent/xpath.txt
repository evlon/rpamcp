## 🔔 新留言检测（双重机制）

### 方式1：检查"有更新(N条)"提示 ⭐ 推荐
**适用场景**: 刷新页面后，第一个文章可能没有红点，但有此全局提示

```javascript
// 精确选择器（xpath.txt 提供）
$('div.filter-bar li.comment-dropdown-item div.weui-desktop-tooltip__wrp span.weui-desktop-form__dropdown__value').html()
// 输出: '有更新(2条)'

// 提取数字
const updateText = $('div.filter-bar li.comment-dropdown-item div.weui-desktop-tooltip__wrp span.weui-desktop-form__dropdown__value').html();
const match = updateText.match(/有更新\((\d+)条\)/);
if (match) {
    const count = parseInt(match[1]); // 2
}
```

### 方式2：检查红点标记
**适用场景**: 首次进入页面或未刷新的情况

```javascript
// 检查是否存在红点
$('div.article-list__item span.article-list__item-badge').length > 0

// 提取红点中的数字
$('div.article-list__item span.article-list__item-badge span').text()
// 输出: '1'
```

## 🎯 完整采集流程

### 1. 进入留言管理页
```javascript
$('ul.weui-desktop-sub-menu > li.weui-desktop-sub-menu__item span:contains("留言")').click();
```

### 2. 检测新留言（双重机制）
```javascript
// 方式1: 检查全局更新提示
const $updateText = $('div.filter-bar li.comment-dropdown-item div.weui-desktop-tooltip__wrp span.weui-desktop-form__dropdown__value');
const hasGlobalUpdate = $updateText.html() && $updateText.html().includes('有更新');

// 方式2: 检查红点标记
const $articlesWithBadge = $('div.article-list__item span.article-list__item-badge');
const hasBadge = $articlesWithBadge.length > 0;

// 任一条件满足即有新留言
const hasNewComments = hasGlobalUpdate || hasBadge;
```

### 3. 遍历文章并采集
```javascript
$('div.comment-articles div.comment-article-list div.article-list__item').each(function(index) {
    const $article = $(this);
    const title = $article.find('div.article-list__item-title').text().trim();
    const hasBadge = $article.find('span.article-list__item-badge').length > 0;
    
    // 处理逻辑...
});
```

### 4. 点击进入文章详情
```javascript
$article.find('div.article-list__item-title').click();
```

### 5. 采集公众号留言者
```javascript
$('div.comment-list__item-container span.comment-nickname').each(function() {
    const $nicknameSpan = $(this);
    const nickname = $nicknameSpan.find('span').first().html();
    const isPublicAccount = $nicknameSpan.find('i.icon-public-account').length > 0;
    
    if (nickname && isPublicAccount) {
        console.log('发现公众号:', nickname);
    }
});
```

### 6. 返回列表页
```javascript
$('.back-btn, .weui-desktop-btn_back').first().click();
```

## 📊 关键选择器总结

## 💡 注意事项

1. **双重检测机制**: 
   - 优先检查"有更新(N条)"全局提示
   - 其次检查每篇文章的红点标记
   - 两者任一存在即表示有新留言

2. **刷新后的行为**:
   - 刷新页面后，第一个文章的红点会消失
   - 但"有更新(N条)"的全局提示仍然存在
   - 因此需要同时检测两种方式

3. **公众号识别**:
   - 通过 `i.icon-public-account` 元素判断
   - 存在即为公众号，不存在为普通用户

4. **XPath vs CSS选择器**:
   - 本文档使用 jQuery CSS 选择器（更易用）
   - 如需 XPath，可使用 `$x()` 函数或 document.evaluate
