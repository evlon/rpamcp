## 🔧 如何调整速度

如果您觉得还是太快或太慢，可以修改以下配置：

### 加快一点（但仍保持拟人化）
```javascript
const CONFIG = {
    MIN_DELAY: 1000,      // 从1500降低
    MAX_DELAY: 2000,      // 从3000降低
    CLICK_DELAY_MIN: 2000,
    CLICK_DELAY_MAX: 3500,
    // ... 其他保持不变
};
```

### 更慢一点（更像真人）
```javascript
const CONFIG = {
    MIN_DELAY: 2000,      // 从1500增加
    MAX_DELAY: 4000,      // 从3000增加
    CLICK_DELAY_MIN: 3000,
    CLICK_DELAY_MAX: 5000,
    // ... 其他保持不变
};
```

### 调整特定行为频率

```javascript
// 在 humanDelay 函数中
if (Math.random() < 0.5) {  // 从0.3增加到0.5，更频繁思考
    
// 在 humanScroll 函数中

// 在文章循环中

// 额外休息
```
